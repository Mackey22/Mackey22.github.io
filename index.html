Honeypot

Spring '15 | C

A honeypot is security mechanism used to track and prevent unwanted packets from corrupting a system. This implementation uses a core to collect packets in a ring buffer and classify them, then send them out to another core that determines if that packet has already been seen. This honeypot just tracks how many of each have been seen.
